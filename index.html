Table CheatSheet

Table Tags

Tag Name Description
<table> Table Create a table
<tr> Table row Create a row
<td> Table data Contain the data of the cell
<thead> Table Head Is a semantic tag that contain the titles of the table
<th> Table heading element Contain the data of the cell to the <thead>.
<tfoot> Table footer element Is a semantic tag that have cells that used to summarize the content of the rest of rows of the table.

Tag Name Description

colspan Column span The column span the space of the row below
